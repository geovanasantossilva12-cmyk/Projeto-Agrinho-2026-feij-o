# Feijão no Brasil

## Sobre o Projeto

Este repositório reúne informações sobre a história, produção, consumo e importância nutricional do feijão no Brasil.

## Objetivos

* Divulgar conhecimento sobre o feijão.
* Reunir dados e estatísticas.
* Disponibilizar material educativo.
* Servir como fonte de pesquisa para estudantes e professores.

## Conteúdo

* História do feijão no Brasil
* Principais variedades
* Benefícios nutricionais
* Produção agrícola
* Consumo nacional
* Curiosidades

## Estrutura

* `docs/` → Documentação principal
* `data/` → Bases de dados
* `assets/` → Imagens e diagramas
* `references/` → Referências bibliográficas
* `presentations/` → Materiais de apresentação
